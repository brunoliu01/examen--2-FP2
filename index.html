examen de fundamentos de programación

1) qué es POO?

La programacion orientada a objetos es una parte de la programación que usa objetos en sus interacciones. Está basada en técnicas, incluyendo herencia, cohesión, abstracción, polimorfismo, acoplamiento y encapsulamiento.

2)explique la diferencia enre clase y objeto.

las clases definen las características de un objeto y el objeto es una instancia de una clase

3)qué es git y github. explique la diferencia.

git es un software de control de versiones de nuestro cógido realizado, por otra parte, github es la herramienta que ofrece dicho servicio a los usuarios

4)qué es heroku?

heroku funciona para poder ver los proyectos realizados en tiempo real.
